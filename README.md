# PIXELS

## Начало работы и сборка

Pixels — это простая 2D-игра на C++ с элементами «выживания» и «песочницы». Игрок управляет персонажем, стреляет по врагам и взаимодействует с цветными квадратами на экране.

### Системные требования

*   [Git](https://git-scm.com/) (для клонирования репозитория)
*   [Библиотека SFML](https://www.sfml-dev.org/download/)
*   [Система сборки CMake](https://cmake.org/download/)
*   Компилятор C++

### Установка и сборка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/nuokey/pixels

1.  **Вписать в терминал следующие команды:**
    ```bash
    cd pixels
    ```
    ```bash
    mkdir build
    ```
    ```bash
     cd build
    ```
     ```bash
     cmake ..
     ```
     ```bash
     cmake --build
    ```
## Структура проекта
main.cpp - основная программа
Папка include - заголовки классов
Папка SRC - тела классов
classes.h - глобальные переменные
CMakeList.txt - сборка всего проекта
